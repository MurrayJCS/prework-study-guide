HTML

  • HTML elements build the structure of our page.
  • The head element contains information about the webpage.
  • The body element represents the content visible to the user.
  • Doctype tells the browser what time of document to expect.
  • The title document defines the title of the webpage and is very important for SEO.

CSS

  • CSS gives our page its style.
  • There are 3 types of CSS: Inline, Internal and External.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • DRY Coding = Don't Repeat Yourself Coding!

Git

  • Git is a free, open-source version control system.
  • Git is responsible for everything GitHub related that happens locally.
  • Git commands start with "git" and are entered into the CLI.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a programming language that allows us to interact with a website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The return function tells the browser to return the result as a variable for later use.
  • Remember JavaScript is case sensitive!
